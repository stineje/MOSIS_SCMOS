Release Notes

  • There is a BUG in this ascii file which can easily be corrected: /import/cad2/osu_stdcells/flow/ami05/encounter.conf
    The line
    set rda_Input(ui_leffile) "$OSUcells/ami05/lib/osu05_stdcells.lef"
    has to be corrected to
    set rda_Input(ui_leffile) "$OSUcells/lib/ami05/lib/osu05_stdcells.lef"

New for Version 2.4

  • Library: No modifications, identical to v2.3
  • Flow: Removed all hard-coded path names
    New variable: $OSUcells
    See flow/README.install for simplified setup procedure
  • Reference Designs: Updated regression testing flow to use $OSUcells

The release is split into 3 parts

  • lib: The actual library files for all 4 technologies
  • flow: The scripts and documentation for the IIT ASIC flow
  • ref_designs: Reference Designs implemented in various tools on all technologies
